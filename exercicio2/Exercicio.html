O que aprendemos

  • MARCAÇÕES (TAGS)

    • A TAG < title> se utiliza para colocar o título; A TAG < p> serve para começar algum paragrafo;
    A TAG < br> serve para quebra de linhas;
    A TAG < h1> serve para definir o tamanho da fonte do titulo secundário;
    A TAG < stlye> serve para definir cores das fontes selecionadas; A TAG < ! - -> utiliza-se parar realizar comentários;

  • IMAGEM E CSS

    • As < TAG max-width e max-height>, mexem no tamanho daa imagem sem alterar a proporção da imagem;
    A TAG < border-radius>, arredonda as bordas da imagem;
    A TAG body é usada para definir o corpo do texto;
    Um cabeçalho TAG define uma parte introdutória da página;
    A TAG link é usada para juntar fotos ao código;
    A TAG img é usada para introduzir uma foto ao código;
