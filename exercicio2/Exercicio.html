  • MARCAÇÕES (TAGS)

    A tag < br> serve para quebra de linhas
    A tag < h1> serve para definir o tamanho da fonte do titulo secundário
    A tag < stlye> serve para definir cores das fontes selecionadas A tag < ! - -> utiliza-se parar realizar comentários

    • As < tags max-width e max-height>, mexem no tamanho daa imagem sem alterar a proporção da imagem
    A tag < border-radius>, arredonda as bordas da imagem
